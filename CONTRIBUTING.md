# Contributing

This is a personal project but contributions are welcome.

## Setup

```bash
git clone https://github.com/pranaya-mathur/LLM-Observability.git
cd LLM-Observability
pip install -r requirements.txt
```

## Making Changes

1. Fork the repo
2. Create a branch
3. Make your changes
4. Test that it works
5. Submit a PR

## Code Style

Try to follow PEP 8. Not too strict about it though.

## Testing

Run `pytest` before submitting. Some tests might fail - that's a known issue.

## Questions

Open an issue if you have questions.

# Architecture Decision Records (ADR)

This folder contains production hardening ADRs for regulated deployments.

## Scope

The ADR set maps key design decisions to controls relevant for:

- India DPDP Act 2023 (privacy, minimization, auditability)
- RBI and SEBI regulated AI governance expectations (risk controls, traceability)
- SOC 2 trust criteria (security, availability, confidentiality)
- Zero-trust security posture (strong authn/authz, secret handling, least privilege)
